> 大概是想记住现在的自己吧，萌发了写年终总结的想法。25年对我而言应该是重启，从24年11月开始, 代入25年的视角，写下这些文字。

去年11月，人生第一次被裁。10月份得知公司可能裁员的消息，当时我们在闲暇时常讨论，如果裁到自己了准备干什么。

11月14号，通知裁员，15号最后一天。下班后和同事简单吃了个饭，回到出租房才真切感受到，裁员了，并且是我。

## 割裂

现在回想起来，25年也是很割裂的一年。一段时间过的很闲暇，一段时间过的很忙碌。

将时间线拉回24年，11月的末尾，我去了北京。在故宫、长城、国博、天安门都留下了景点游客照。

12月我去了曼谷，稍微丰富了下自己的人生。

全世界最大的成人乐园 / 近距离接触狮子 / 曼谷国家博物馆 / 湄南河夜景 / 王权大京都大厦 / 海军射击场 / 鱿鱼游戏宣传

返程时在澳门中转，拜了妈祖，去了赌场，看了圆明园兽首特展。

这段时间也尝试过找工作，但就业难度比我预想的要高。当自己的预期与现实一次次碰撞，我开始反思自己，我开始尝试改变。

之前一直觉得有了工作经验，肯定不会再去询问一些“八股文”。我理解的面试是深入项目，考察解决问题的能力。但当时自己的状态很差，项目理解停留在表层，尽管这些项目都是我曾经实现的却不能很好的表达出来。

24年末尾，25年年初。

春节即将到来，我很明显感觉到 HC 开始收缩。这段时间我开始利用豆包辅助我梳理面经，模拟面试。整理了一些常见的“八股文”、深入自己的项目，了解库包实现原理以及一些常见的面试话术。

失业的这段时间里，我时常感到孤单。朋友们都有自己的工作要忙，想出去玩往往找不到人。于是我开始沉迷电子游戏，用游戏打发时间。这段时间我玩了死亡搁浅、文明6以及永劫无间，死亡搁浅通关了，文明6我记得似乎玩了70～80个小时，永劫无间每天都在玩。当然中间也穿插了别的游戏，但是已经没什么印象了。

过完年，3月了。周二找几家公司投递简历，周三到周五面试，这是我当时的策略，不想给自己太多压力。

面试了一家图形编辑器公司，很明显的重前端项目。和自己的以往的项目经验相似，拉扯了近半个月的时间，已经到了终面，最后不知道为什么面试终止了，很郁闷。之后面试了一家游戏公司，我自己本身没有做游戏的经验，感觉可能又是走过过长。面试的时候，得知项目组是做游戏辅助工具的，之后的工作是负责官网、SDK以及运营后台开发。

在面试这家游戏公司时，我当时的状态是比较差的，接连的失败让我很受打击，想着如果再失败的话，出去玩一段时间散散心吧。然后就拿到 offer 了，准备上班。

入职的流程也比较麻烦，体检、背调、下发 offer，最后培训。

往年的体检基本全绿，这次入职体检发现了两个问题，肾结晶以及右眼近视，索性不是什么大问题，只能说失业这段时间还是过于放纵自己了。

- 很少喝水
- 经常关灯玩电脑

## 重启

2025年4月7号，入职。

刚入职的那几天，我曾想过放弃这份工作。项目组处于一个亏损状态，原先说负责的众多项目最后变成了只有运营后台，只有 window 开发环境。我记得当时自己似乎挣扎了一段时间，但是此刻的我已经忘记当时是如何劝自己留下的了。

除此之外还有很多槽点

- 没有自己的 git 账号 (开发时经常会遇到多个需求并行开发，有时封装好的工具函数在两个需求中都要使用，这时候我一般到 git 平台复制)
- 离谱的 bug 管理系统 (没有主动通知机制，需要自己去平台上刷新，经常忘记检查，导致一些 bug 没有及时修复)
- 严格的权限管理机制 (限制外网 git 权限，以及云文档访问权限)
- 发工资遇到非工作日会延后

还有一些我觉得看个人情况的槽点，对我而言，算是好事。

- 加班比较多 (项目组不加班的话，会被约谈，但是耐不住有加班费)
- IM 工具是 QQ (没有已读功能)

平淡的日子结束的很快，游戏辅助工具，本质上是一个 toC 的产品。游戏公司，采用游戏的开发模式也很合理，所以我们的游戏工具是一个联运项目。

联运方应该是感觉亏损太多了，主动介入运营，制定了一系列开发计划。从6月开始，我们开始UI重新设计，迭代福利活动。我开始接管 APP H5 福利页开发。

运营后台的迭代并没有太多的难度，但在接手 toC 业务后就很忙碌了。但相对来说我也积攒了一些项目经验，对我个人来说是好的吧。

有的时候不得不感慨别人的专业，在这一系列操作下，日活翻了 5 倍，流水翻了 20 倍。终于不要担心如果一直处于亏损状态，会不会裁员了。

至此，25年的工作也到这里结束了。

在上家公司，我很拒绝使用 AI 生成代码，总感觉自己手写的代码会比 AI 生成的好。一般使用 Chat 模式，让 AI 协助搜索文档，解决问题。

但是 Claude、GPT、Gemini 这些模型迭代的太快了，有点超乎我的想象。于是我开始尝试使用 AI 生成代码，我专心负责需求梳理以及 Code Review 。

在12月份，我的部分需求已经完全交个 AI 处理了，这导致我对未来很悲观，感觉自己很容易被替代。

## 平淡的日子

今年的生活其实没有什么特别的，没有一丝波澜。

因为加班，现在周末也很少出去玩了，为数不多的出门还是在杭州重复往年的景点。去的最多的是天下第一财神庙，祈求发财🙏

朋友们陆陆续续结婚了，开始经营自己的小家。几个朋友小孩出生了，想来以后见面的机会更少，关系已经止步在微信上偶尔闲聊几句。

其实上班后除了同事就很少认识新朋友，但是今年似乎有点改变了，在游戏中认识了几个人，在小区二手交易群认识了几个人。虽然不是现实生活中的朋友，但是也算是一点进步吧。

永劫无间今年占据了我很大的一部分时间，我其实没什么游戏天赋，花了比较多时间，达到了游戏中的最高段位，也算是一点收获吧。但是我已经感觉到自己对游戏热爱的减少，也许做完这个赛季任务，就会慢慢退游了吧。

下半年看到很多人使用 AI 做了很多很有意思的产品，打破了只有程序员才能开发的限制，我不禁感叹 AI 对生产力的提升。而我自己仿佛困在了 web 的世界里，无法跳出这个怪圈，至今仍在尝试开发一个的全栈项目。

## 尾声

今年的我似乎成熟了一些，但是相比较我现在的年纪，我的成熟应该再早几年开始就好了，蹉跎时光。

我其实很少规划未来，打算在 26 年尝试做一些改变。

- 英语一直是我的弱项，打算每天背几个单词，日积月累
- 尝试把目前正在进行的全栈项目开发完成，跳出怪圈

其他的一切，随缘吧。
